# GitHub Achievements Plan

A step-by-step plan to earn all GitHub profile achievements.

---

## Achievements Checklist

### 1. Pull Shark
- [ ] Create a feature branch
- [ ] Make changes and open a Pull Request
- [ ] Get the PR merged
- [ ] Repeat to unlock higher tiers (x2, x4, x8, x16)

### 2. YOLO
- [ ] Open a Pull Request
- [ ] Merge it without requesting a code review

### 3. Quickdraw
- [ ] Open an issue or Pull Request
- [ ] Close it within 5 minutes of opening

### 4. Pair Extraordinaire
- [ ] Make a commit using the `Co-authored-by` trailer
- [ ] Example:
  ```
  git commit -m "feat: description

  Co-authored-by: Name <email@example.com>"
  ```
- [ ] Repeat to unlock higher tiers (x2, x4, x8)

### 5. Starstruck
- [ ] Make repository public
- [ ] Share repo link to get stars
- [ ] Milestones: 16, 128, 512, 4096 stars

### 6. Galaxy Brain
- [ ] Participate in GitHub Discussions
- [ ] Have 2+ answers marked as the accepted answer
- [ ] Repeat for higher tiers (x2 → 8 answers, x8 → 16, x16 → 32)

### 7. Public Sponsor
- [ ] Sponsor an open source developer via GitHub Sponsors

### 8. Heart On Your Sleeve
- [ ] React to any issue, PR, or discussion using the ❤️ emoji

### 9. Open Sourcerer
- [ ] Have PRs merged in multiple public repositories (not just your own)

## Tips

- Most achievements have **multiple tiers** — keep repeating to unlock higher levels.
- Achievements are visible on your **GitHub profile page**.
- Some achievements are **legacy** (Arctic Code Vault, Mars 2020) and can no longer be earned.
- Make sure your **profile is public** for achievements to show.
- **Galaxy Brain** requires at least 2 accepted answers to unlock the base badge.
- **Heart On Your Sleeve** may be in beta and limited to certain accounts.
